# 🎮 Valorant Tier 2 → Tier 1 Scouting Analytics Dashboard

📊 End-to-end Power BI decision system for evaluating Tier 2 players readiness for Tier 1 competition using fully data-driven analysis and model-based scoring.

---

## 📌 Overview

This project is a **data-driven scouting decision system** designed to evaluate whether Tier 2 Valorant players are ready to be promoted to Tier 1 competition.

Key characteristics:
- No predefined rankings in the dataset
- No subjective labels or scouting bias
- All insights are fully model-driven

👉 Rankings and conclusions are entirely derived from analytical logic.

---

## 🎯 Business Problem

Professional teams must answer a critical question:

**Are there Tier 2 players truly ready for Tier 1 competition?**

Challenges include:
- Objective performance evaluation
- Avoiding bias and reputation-based decisions
- Ensuring data-driven selection

---

## 📊 Data Source

- Source: VLR.gg
- Scope: Tier 2 (EMEA)
- Sample: 66 players

Dataset characteristics:
- No rankings or labels
- No qualitative insights
- Pure performance metrics

---

## 🧠 Analytical Approach

Players are evaluated across four dimensions:

1. Performance (Impact)
2. Readiness (Benchmarking vs Tier 1)
3. Reliability (Sample size proxy)
4. Efficiency & Consistency

---

## 🧮 Scoring Model

Final Score =
(Readiness * 0.35) +
(Impact * 0.20) +
(Reliability * 0.15) +
(Kill Efficiency * 0.10) +
(Impact per Round * 0.10) +
(Consistency * 0.15)

---

## 📊 Dashboard Structure

🟢 Executive Overview → What is happening

🔵 Diagnosis → Why it is happening

🔴 Action → What should be done

---

## 📈 Key Insights

- Only ~6% of players approach Tier 1 level
- No player is fully ready for promotion
- Main limitation: inefficiency and inconsistency
